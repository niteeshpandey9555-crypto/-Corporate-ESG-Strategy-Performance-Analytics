## 🎯 How to Import Data

### Method 1: Excel Import Wizard
1. Open Excel Dashboard
2. Go to Data → Get Data → From File → From Text/CSV
3. Select CSV file
4. Map columns to existing sheet structure

### Method 2: Copy-Paste
1. Open CSV in Excel
2. Select all data (Ctrl+A)
3. Copy (Ctrl+C)
4. Paste into appropriate dashboard sheet

### Method 3: Power Query (Advanced)
1. Data → Get Data → From File → From Folder
2. Select folder with all CSVs
3. Combine and transform data
4. Load to dashboard
